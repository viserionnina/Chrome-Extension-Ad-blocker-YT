# Chrome Extension - AD blocker for YT
- This project is a YouTube ad blocker where we are have Chrome extension using Manifest V3.
- By utilizing specific CSS selectors that YouTube uses (e.g. declarativeNetRequest) it can intercept and block network requests to known ad-serving domains (e.g. doubleclick.net). 
- The project contains a manifest.json file that is like a id card for what this extension will do. For configuration and a rules we have backgorund.js and to define and to block certain content content.js.

## To use:
1. Go to: chrome://extension in Google Chrome
2. Toggle on Develepor mode to ON.
3. Click Load unpacked and select your project folder.
